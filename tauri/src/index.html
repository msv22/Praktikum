Импорт курса из JSON

Вставьте JSON курса или выберите файл. Перед добавлением он пройдёт проверку структуры.

Формат JSON курса

Минимальный рабочий пример со всеми типами шагов. Поле homework — обязательное ДЗ урока: без него урок не засчитается автоматически.


    

      
type: "text"теория; content — Markdown (заголовки #, списки, **жирный**, `код`, блоки ```)

      
type: "quiz"options — варианты, correct — индексы верных (с нуля); несколько индексов = мультивыбор; explanation — пояснение после ответа

      
type: "input"answer — ответ (или answers — массив вариантов); match: "exact" (по умолч.), "contains", "number" (+ tolerance)

      
homeworkмассив шагов quiz/input в конце урока; отмечается отдельно и обязателен для авто-зачёта

    

    

  






  

    
Сгенерировать курс (AI)

    
Работает с любым OpenAI-совместимым API: DeepSeek, OpenAI, Gemini, OpenRouter, локальная Ollama. Ключ хранится только на этом устройстве и уходит напрямую провайдеру.

    

      

    

      

      

    

    

    

      

        

      

        

    

    

      

    

    
Курс собирается по частям: сначала план, потом каждый урок отдельным запросом с проверкой формата — надёжнее, и виден прогресс. Если провайдер ответит «модель не найдена» (404), впишите актуальное имя модели вручную — поле редактируемое.
