# Contributing to Praktikum

Спасибо за интерес к проекту! Правки, идеи и баг-репорты приветствуются.

## Как устроен код

Всё приложение — один файл `index.html`:
- секция `<style>` — дизайн-система (переменные `--*`, тёмная «звёздная» тема);
- секция `<script>` — вся логика;
- словарь локализации `I18N` в начале скрипта;
- строгая валидация курса — функция `validateCourse()`;
- AI-клиент (OpenAI-совместимый) — `generateCourseWithAI()` и `callChat()`.

Сборки не требуется: откройте `index.html` в браузере и правьте.

## Как предложить изменение

1. Сделайте форк и ветку: `git checkout -b feature/название`.
2. Внесите правки в `index.html`.
3. Проверьте вручную: генерация, импорт JSON, прохождение урока, смена языка.
4. Откройте Pull Request с кратким описанием.

## Что особенно приветствуется

- **Переводы.** Улучшение 简体中文, добавление новых языков (скопируйте блок в `I18N`).
- **Курсы.** Готовые `.json`-курсы в папку `courses/` — с указанием темы и языка.
- **Новые типы шагов.** Например, шаг «код с автопроверкой» (Pyodide).
- **Светлая тема.**

## Стиль

- Без внешних зависимостей в рантайме — проект должен работать офлайн из одного файла.
- Никаких секретов в коде: API-ключи вводит пользователь, они не коммитятся.
- Комментарии и UI — на понятном языке; строки интерфейса — только через `t()`.

## Лицензия

Отправляя PR, вы соглашаетесь, что ваш вклад распространяется под [MIT](LICENSE).
